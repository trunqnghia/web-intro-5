# web-intro-5 Tóm tắt Vai trò Internet & Web

Tổng Quan Về Internet Và Thế Giới Web

Tóm tắt kiến thức cốt lõi cho Bài 1

1. Internet là gì?

Internet là một mạng lưới toàn cầu kết nối hàng triệu máy tính và thiết bị điện tử với nhau. Nó cho phép người dùng chia sẻ thông tin và giao tiếp từ bất kỳ đâu trên thế giới.

Vai trò: Làm nền tảng cơ sở hạ tầng cho các dịch vụ khác chạy trên nó (như Web, Email, Chat, File transfer...).

2. Web (World Wide Web - WWW)

Web là một không gian thông tin toàn cầu, nơi các tài liệu và tài nguyên mạng khác được định danh bằng các URL và liên kết với nhau qua các siêu văn bản (hypertext).

Lưu ý phân biệt: Internet là "đường ống dẫn nước" (phần cứng), còn Web là "nước" chảy trong đường ống đó (phần mềm/dữ liệu).

3. Website (Trang web tổng thể)

Website là một tập hợp các trang web liên quan đến nhau, thường nằm chung một tên miền (domain) và được lưu trữ trên cùng một máy chủ (hosting).

Ví dụ: google.com, wikipedia.org là các website.

4. Trang web (Webpage)

Trang web (Webpage) là một tài liệu đơn lẻ hiển thị trên trình duyệt web. Một website được cấu thành từ nhiều trang web khác nhau.

Các thành phần cơ bản của một trang web thường bao gồm:

  • Nội dung văn bản (Text), Hình ảnh (Images), Video.
  • Các liên kết (Links) dẫn tới trang web khác.
  • Các thẻ định dạng HTML/CSS để tạo giao diện.
